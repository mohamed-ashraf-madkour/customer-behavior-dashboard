# 📊 Customer Behavior Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing customer behavior using SQL, Python, and Power BI to uncover valuable business insights related to customer purchasing patterns, revenue generation, demographics, and product performance.

The project combines data cleaning, exploratory data analysis (EDA), SQL querying, and interactive dashboarding to support data-driven business decisions.

----------------------------------------------------------------------------------------------------------

# 🛠️ Tools & Technologies
* Python
* SQL Server
* Power BI
----------------------------------------------------------------------------------------------------------

# 📂 Project Workflow

## 🟦 1. Data Cleaning & Preparation (Python)

Performed data preprocessing and cleaning using Python.

### Tasks:
* Handling missing values
* Removing duplicates
* Feature engineering
* Creating age groups
* Creating price categories
* Data formatting and transformation
--------------------------------------------------------------------------------------------------------

## 🟩 2. SQL Analysis

Used SQL Server to explore customer behavior and generate business insights.

### Analysis Includes:

* Total revenue analysis
* Male vs Female purchasing behavior
* Customer segmentation
* Product category analysis
* Average customer spending
* Revenue by age group
* Top-performing products

### SQL Concepts Used:

* Joins
* Aggregations
* Group By
* CTEs
* Views

----------------------------------------------------------------------------------------------------------

Built an interactive dashboard to visualize customer insights and business performance.

### Dashboard Includes:

* Revenue Overview
* Customer Demographics
* Gender Analysis
* Product Performance
* Sales Trends
* Customer Segmentation
* Interactive Filters & Slicers

### KPIs:

* Total Revenue
* Total Customers
* Average Spending
* Best-Selling Categories
* Customer Distribution

--------------------------------------------------------------------------------------------------------

# 📈 Key Insights

* Female customers generated higher overall revenue.
* Certain product categories significantly outperformed others.
* Middle-aged customers represented the highest spending segment.
* Customer purchasing behavior varied across age groups and genders.
* High-priced products generated strong revenue despite lower sales volume.
